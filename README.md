# DrawPerfectCyrcle
## Requirements

 - Eclipse Adoptium\jdk-11.0.15.10-hotspot
 - OS Windows 10
 - Chrome or Chromium
## How to use?
 - Download the `DrawPerfectCyrcle.jar` from the release section.
 - Go to a shell and enter `java -jar DrawPerfectCyrcle.jar`
 - Follow the instructions on the shell.
